JFTT - Kompilator - Czn

Kompilator został napisany w Pythonie przy użyciu SLY.

Kompilator składa się z:

README.txt - pliku z opisem
Logic.py - pliku zawierającego logikę
Models.py - pliku ze strukturami danych
MyLexer.py - pliku zawierającego analizę leksykalną kodu wejściowego
MyParser.py - pliku zawierającego parser języka
kompilator.py - pliku uruchamiającego kompilator

Wymagania:
Python 3.8.5
sly 0.4

pip3 install sly
